Release Date

MyBB 1.4 was released on 3 August 2008.

Announcement Summary

I'm very proud to announce that MyBB 1.4 has been released and is now available directly on the MyBB website.

MyBB 1.4 is a great leap forward in terms of both functionality of the board as well as a complete overhaul of the administrator control panel. There are over 70+ new features that have been built in to MyBB and the MyBB code base has almost doubled in size since MyBB 1.2.

More Information: http://community.mybb.com/thread-34565.html

Overview

MyBB 1.4.0 is the latest major release for the MyBB series, meaning it will includes several new features and other tweaks to improve the MyBB software.

A feature tour of what's new in MyBB 1.4 is available on the MyBB website, as well as a complete new feature listing.

Confirmed Features and Changes

The following is a list of confirmed features and changes (either through a developer's blog, through the MyBB Community Forums, or through the MyBB Ideas site) for MyBB 1.4.0. This is by no means a complete listing, and is shown here for archival purposes.

  • A new administrator control panel
    • Complete Overview of the MyBB 1.4 Admin CP on the MyBB Development Blog:
      • Part 1 of 4 (Introduction, Home, and Configuration)
      • Part 2 of 4 (Forums & Posts, Tools & Maintenance, and Design Decisions)
      • Part 3 of 4 (Users & Groups and Technical Decisions)
      • Part 4 of 4 (Templates & Style and Questions & Answers)
    • A MyCode Tester has been added
  • Multiple PM recipients with BCC (Blind Carbon Copy) support
  • AJAX Quick Reply with:
    • New posts detection
    • New page detection (based on your settings)
  • Automatic search engine friendly URLs (more information)
  • Search this thread
  • Administrators and Moderators can see who voted for what on "Show Details" page regardless of whether it is a private or public poll.
  • Language selection for guests
  • SMTP mail function
  • Automatic avatar resize (more information)
  • Updated Thread Subscriptions (more information)
    • Removal of "Favorite threads"
    • Ability to have no notification or instant email notification
  • Existing backups will be sorted by date.
  • MyCode editor on the "Edit Signature" page
  • Plugin update checking (more information)
  • AJAX validation on registration
  • Updated quote tag (links back to original post & includes time of the post)
  • Automatic DST detection
  • Additional PM information: "You replied to/forwarded this message on X"
  • User group post/rank/registration time based promotions/demotions
  • PM pop-up Removed - Replaced with something else
  • Automatically detects proper cookie information on installation
  • New forum/thread marking system
  • Calendar rewrite, including multiple calendars, recurring events, user group based permissions, and moderator controls (more information)
  • Buddy List support for Private Messages (more information)
  • Location indicator on profile (more information)
  • Revamped buddy list popup (more information)
  • Task system (more information)
  • SQLite 2 & 3 and PgSQL support (more information)
  • True slaves/masters load-balancing system for MySQL(i) and PgSQL (more information)
  • A moderator control panel (more information)
  • COPPA compliance
  • New smilies
  • New theme system allowing you to specify multiple stylesheets per theme and tell MyBB which pages or actions they should be attached to
  • Warning system
  • New (optional) horizontal style post layout (see screenshot)
  • Redesigned User CP (see screenshot)
  • New editor interface, which can sport an Office 2007 style interface (see screenshot - see screenshot)
  • New buddy and ignore list manager, using AJAX (see screenshot)
  • Revamped member list
