This is obviously not the complete feature list. See Features.


This template is used on pages that describe features of MyBB. Usually these are pages that link to other pages in the documentation, and are "hub" pages.

For example, user groups are mentioned in various places in the documentation, including the User CP and Admin CP. This "feature" page will be the place where user groups can be described in general. Then links can also be provided to the User CP, Admin CP, etc.
